# chrome-noise-gate
A proof of concept Chrome Extension that lets you dial in a custom noise gate for any streamed video. The goal is to help those with Misophonia, or those who just don't want to hear the sounds of someone eating etc, watch videos with such content.


## Steps to install
- Download and extract code from repo
- In Chrome, navigate to <a href="chrome://extensions" target="/">chrome://extensions</a>
- Click "Load unpacked"
- Navigate to where the Zip was extracted

## Custom Preset
To set the custom preset simple long press it for at least 600ms. This will either save or overwrite the saved preset with the current slider values

Once I'm more confident that this works I'll submit it to the Chrome store and maybe port it to other browsers.
